## 🧠 About Me

Advanced ML Engineer with **3+ years** building end-to-end machine learning systems in production — from raw data to deployed models.

I've worked across **computer vision**, **scientific ML research**, and **optimization systems**, leading teams and contributing to peer-reviewed research in collaboration with Italian universities.

Outside of production ML, I explore the evolving world of **LLMs, AI agents, and RAG architectures**.

<br/>

- 🤖 **Computer Vision** — object detection & classification with YOLO
- ⚙️ **MLOps** — automated pipelines with Airflow, experiment tracking with MLflow, versioning with DVC
- 🧪 **Research** — molecular property prediction for CO₂ catalysis *(paper in submission)*
- 🔌 **Backend** — FastAPI microservices, PostGIS, SQL
- 🌱 **Exploring** — LLMs, AI agents, RAG, GenAI in production
- 🎓 **Education** — MSc Computer Engineering, AI Curriculum · 110/110 *cum laude* · UniSalerno
